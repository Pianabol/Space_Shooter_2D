#  2D Space Arcade (Learning Project)

*A foundational 2D arcade shooter built with Unity. This project serves as an archive of my early steps into game development, 2D physics, and Unity engine mechanics, originally developed alongside a Udemy course.*

##  Features & Mechanics

Despite being an introductory project, it includes core arcade loops and a structured progression system:

*   **Level Progression:** 11 fully playable levels. Each new level is strictly locked and only becomes available after successfully clearing the previous one.
*   **Highscore System:** Tracks and saves the player's highest score across sessions.
*   **Enemy Variety:** Defend against incoming Meteors, agile Pink & Green enemy spaceships, and face off against a challenging end-game Boss.

##  Power-Ups

The game features a dynamic drop system to help you survive the chaos:

*   🛡️ **Shield:** Grants absolute protection for 3 consecutive hits. While active, the shield acts as a battering ram, instantly one-shotting any enemy it touches.
*   🔫 **Weapon Upgrade (Stackable):** Upgrades your firepower progressively. Collecting multiple shooting power-ups increases your simultaneous laser output: `1 ➔ 2 ➔ 3 ➔ 5 ➔ 7` lasers.
*   ❤️ **Health Pack:** Instantly restores 2 health points/lives to keep you in the fight.
